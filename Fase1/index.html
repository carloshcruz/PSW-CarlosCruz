Principales Etiquetas HTML

# Etiqueta Función
1 <aside>…</aside> Define el contenido lateral del contenedor de una página
2 <body>…</body> Define el cuerpo del documento
3 <br>…</br> Define un salto de línea
4 <div>…</div> Define una sección en un documento
5 <footer>…</footer> Define el pie de página de un documento
6 <head>…</head> Define información hacerca del documento
7 <header>…</header> Define la sección de encabezado del documento
8 <li>…</li> Define un ítem de una lista
9 <p>…</p> Define un párrafo
10 <table>…</table> Define una tabla
 

Diferentes Estilos que Pueden Aplicarse en CSS

# Estilo Función
1 colors Establece el color del texto
2 backgrounds Para definir las propiedades del fondo de un elemento
3 borders Para definir las propiedades del borde de un elemento
4 margins Para definir la distancia que debe existir entre un elemento y los elementos que lo rodean
5 text Se usa para establecer el formato de texto a subrayado y suprarrayado, tachado o parpadeo
6 fonts Permite establecer algunas o todas las propiedades tipográficas del texto de forma simultánea
7 height Especifica la altura del area de contenido de un elemento
8 width Especifica la anchura del area de contenido de un elemento
9 links Representa un elemento o enlace que aún no ha sido visitado
10 lists Agrega colores de fondo y propiedades de marcadores de los elementos de una lista
11 tables Representa datos en dos o mas dimensiones; filas y columnas para generar una tabla
12 position Indica la posición del gráfico dentro del bloque
13 overflow Permite regular la visiblidad de los contenidos que sobresalen de una caja html
14 float Ubica un elemento al lado izquierdo o derecho de su contenedor
